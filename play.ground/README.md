## рџђ› Troubleshooting

### WASM fails to load
```bash
# Rebuild WASM
npm run build

# Check files exist
ls -lh xon.js xon.wasm
```

### CodeMirror not loading
- Check internet connection (loads from CDN)
- Or download CodeMirror locally

### Port 8000 already in use
```bash
# Change port in package.json
"start": "http-server -p 3000 -c-1 -o"
```
